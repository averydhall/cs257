Name:
python3 booksdatasource.py -- parses books1.csv

Synopsis:
python3 booksdatasource.py
Arguments:
[-title] Title [-t|-y]
[-author] Author
[-range] Start End
[-usage]


Description:
  python3 booksdatasource.py has 3 main uses (+ a usage statement command) each
  of which parses books1.csv in a given manner (searching by title, searching by author, or searching by range)
  and returns a list of relevant data.

  -title  takes in a string and returns a list of book titles that contain the given string (case-insensitive)
      -t  used in conjunction with -title, will return list of book titles sorted alphabetically
      -y  used in conjunction with -title, will return list of book titles sorted by publication year

  -author  takes in a string and returns a list of authors whose names contain
           the given string, and for each author a list of their books
           (ordered alphabetically by surname, ties are broken by given name)

  -range  takes in a starting year and ending year and returns a list of books
          published between those years (inclusive, books ordered by publication year)

  -usage  returns the usage statement
